# Call Scheduler
Python scheduler with all source files

## hard_coded
  All pre-scheduled call days, ignores anything not on Friday-Sunday
## residents
  basic resident information with availability per block
## incomplete_schedule
  Final schedule to be placed here, if manual modificatons were needed, Call_scheduler_v2 will then use this as a base and re-balance the rest of the year
## priorities
  Request priorities, coded as "soft" and "hard" aka will be nice, and absolute

## Call_scheduler_v2 - my beauty

Will generate an export - call_schedule with 3 sheets, the schedule listed, and the distribution and a calendar output

Note: Date formating YYYY-MM-DD
